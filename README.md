# 🏡 Airbnb Clone (MongoDB Project)

## 📌 Overview
This project is a MongoDB-based Airbnb Clone that allows users to explore, add, and manage property listings. It simulates a real-world accommodation booking platform where users can view listings and hosts can manage their properties.

The project focuses on database design, CRUD operations, and backend logic implementation.

---

## 🚀 Features
- 🏠 Add new property listings
- 📋 View all listings
- ✏️ Update listing details
- ❌ Delete listings
- 🔍 Search and filter properties (if implemented)
- 👤 User-based data handling (optional)

---

## 🛠️ Tech Stack
- MongoDB
- Node.js
- Express.js
- Mongoose (ODM)

---

## ⚙️ How It Works
1. User can add a new property with details like title, price, location  
2. All listings are stored in MongoDB database  
3. Users can view available properties  
4. Listings can be updated or deleted  
5. Backend APIs handle all CRUD operations
